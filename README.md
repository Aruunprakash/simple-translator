<b>📌 Project Description</b>
<br>
Simple Translator is a lightweight Python-based command-line application that translates words or phrases into multiple languages using the googletrans library. It detects the source language automatically and converts the input into a user-specified target language from a predefined set.
This project is designed to demonstrate basic concepts of API usage, input validation, and language processing in Python.

⚙️ Technology Used
<br>
* 🐍 Python
* 📦 googletrans library

✨ Features
<br>
* 🌐 Automatic source language detection
* 🔄 Translation into multiple languages
* ✅ Input validation for allowed target languages
* 🧾 Displays detected and target language names
* ⚡ Simple and user-friendly CLI interface

🔄 Process / How It Works
<br>
The application works by first taking a word or phrase as input from the user along with a target language code from a predefined set of allowed languages. It then validates whether the selected language is permitted, and if valid, uses the googletrans library to automatically detect the source language of the input text. After detection, the program translates the text into the chosen target language and displays the detected language, target language, and the translated output to the user in a simple command-line interface.
